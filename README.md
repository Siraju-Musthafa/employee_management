 Employee Management System

A backend REST API built using Django and Django REST Framework.

Features
- Add employee
- Get employee by ID
- List active employees
- Update employee
- Soft delete employee

Tech Stack
- Python
- Django
- Django REST Framework
- SQLite

API Endpoints
- POST /api/employee/
- GET /api/employee/{id}/
- GET /api/employee/active/
- PUT /api/employee/update/{id}/
- DELETE /api/employee/delete/{id}/

Setup
1. Create virtual environment
2. Install dependencies
3. Run migrations
4. Start server
